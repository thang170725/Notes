Tesseract OCR
    • Loại: OCR truyền thống (không deep learning), rất phổ biến trong các bài toán đơn giản
    • Ưu điểm:
        ◦ Dễ dùng, cài một phát chạy được ngay
        ◦ Không cần GPU
        ◦ Hỗ trợ 100+ ngôn ngữ
        ◦ OCR tốt trên văn bản in, font đẹp
        ◦ Miễn phí, open-source
    • Nhược điểm:
        ◦ Rất kém khi: chữ nghiêng, chữ méo/vòng cung, ảnh nhiễu, ảnh scan xấu, chữ nhỏ, Không tốt với cảnh thật (street view)
    • Dùng khi nào?
        ◦ Trích xuất text từ:
        ◦ hóa đơn rõ ràng
        ◦ tài liệu scan đẹp
        ◦ bảng in, chữ rõ
    • Cần pip install pytesseract
