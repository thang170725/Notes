VietOCR (Transformer cho tiếng việt)
    • Loại: OCR deep learning chuyên Recognition, rất phổ biến ở VN, accuracy cao
    • Ưu điểm:
        ◦ Chính xác tiếng Việt tốt nhất
        ◦ Mô hình nhỏ, chạy nhanh
        ◦ Tùy biến dễ (fine-tune với dữ liệu riêng)
        ◦ Đọc được font xấu, ảnh nghiêng, hóa đơn
    • Nhược điểm:
        ◦ Không có Text Detection (cần model khác để lấy vùng chữ)
        ◦ Phải tự code workflow Detection → Recognition
    • Dùng khi nào?
        ◦ OCR hóa đơn Việt Nam
        ◦ OCR văn bản scan tiếng Việt
        ◦ OCR tài liệu tiếng Việt dài (form, biểu mẫu)
