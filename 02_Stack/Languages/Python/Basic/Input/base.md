**Syn**
```bash
variable = input(prompt)
- prompt (Tùy chọn): Là một chuỗi ký tự (string) hiển thị lên màn hình để thông báo cho người dùng biết họ cần nhập gì. Nếu bạn để trống, người dùng sẽ thấy một màn hình trống và không biết phải làm gì.
- variable: Là biến dùng để lưu trữ dữ liệu mà người dùng nhập vào từ bàn phím.
- Lưu ý quan trọng nhất: Hàm input() luôn luôn trả về kiểu dữ liệu là Chuỗi (String), bất kể bạn nhập vào là số hay chữ.
```
