# []
```bash
Thay đổi giá trị của một mục cụ thể bằng cách tham chiếu đến khóa của nó.
```
**Ex1**
```python
def main():
    di = {
        "name": "John",
        "age": 30
    }
    di["age"] = 10
    print(di)
main()

# {'name': 'John', 'age': 10}
```
**Ex2: Add items**
```python
thisdict =	{
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

thisdict["color"] = "red"

print(thisdict) # {'brand': 'Ford', 'model': 'Mustang', 'year': 1964, 'color': 'red'}
```
