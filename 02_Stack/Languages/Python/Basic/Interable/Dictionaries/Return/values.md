# .values()
```bash
- Trả về danh sách các giá trị của dictionary.
```
**Ex**
```python
data = {
    'names': ['Thang', "Minh", "Nghia", "Quy"],
    'ages': [18, 20, 23, 16],
    'scores': [10, 9, 6, 7]
}

print(data.values()) # dict_values([['Thang', 'Minh', 'Nghia', 'Quy'], [18, 20, 23, 16], [10, 9, 6, 7]])
```
