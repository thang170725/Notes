# list()
Ép dữ liệu sang kiểu mảng.
**Cú pháp**
```text
list(a)
```
**EX**
```python
a = "My namme is "
print(list(a)) 

# ['M', 'y', ' ', 'n', 'a', 'm', 'm', 'e', ' ', 'i', 's', ' ']
```
# len()
Trả về độ dài của một mảng.
**Cú pháp** 
```text
len(<variable>)
```
