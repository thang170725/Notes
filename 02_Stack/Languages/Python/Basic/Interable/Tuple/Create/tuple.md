# tuple()
```bash
- Để ép từ một kiểu nào đó sang tuple.
```
**Syn**
```bash
tuple(<variable>)
```
**Ex**
```python
days = ("Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday", "Monday")
print(days) # ('Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Sunday', 'Monday')
```
