# []
```bash
- Các mục tuple được lập chỉ mục, mục đầu tiên có chỉ mục [0], … (giống mảng).
```
**Ex**
```python
days = ("Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday")
print(days[0])
days[0] = "Thứ 2"
print(days[0])
```
