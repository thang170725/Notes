# []
**Syn**
```bash
[start : end : step]
- start : vị trí bắt đầu (tính từ 0)
- end : vị trí kết thúc (không lấy end)
- step : bước nhảy
```
**Ex**
```python
text = "abcdef"

text[0] # 'a'
text[-1] # 'f'
text[1:4] # 'bcd'
text[:3] # 'abc'
text[3:] # 'def'
text[::] # 'abcdef'
text[::2] # 'ace'
text[::-1] # 'fedcba'
```
