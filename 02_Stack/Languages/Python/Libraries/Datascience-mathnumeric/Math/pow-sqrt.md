# math.sqrt()
```bash
- Để lấy căn bậc 2 của một số.. Thư viện math không cần tải về mà có thể import trực tiếp.
```
**Syn**
```bash
math.sqrt(value)
```
**Ex**
```python
import math
a = math.sqrt(36)
print(a)    # 6
```
