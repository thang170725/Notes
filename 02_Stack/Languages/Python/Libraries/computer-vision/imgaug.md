imgaug
9.5.1 Kiến thức cơ sở
    • imgaug là một thư viện python chuyên để tạo ảnh giả lập (synthetic image) thông qua augmentation mạnh mẽ.
    • Nó rất linh hoạt, có thể thao tác bounding boxes, keypoints, heatmaps, segmentation mask, ..
    • Ưu điểm: Linh hoạt, có thể dùng theo kiểu mô hình hóa thống kê (lambda, uniform(), normal()). Hỗ trợ cả numpyndarray, dễ tích hợp pipeline tùy chỉnh.
Cài đặt môi trường
pip install imgaug
