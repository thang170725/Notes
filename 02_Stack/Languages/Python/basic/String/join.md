# .join()
```bash
Lấy tất cả các mục trong một iterable và nối chúng thành một chuỗi. Một chuỗi phải được chỉ định làm dấu phân cách.
```
**Syn**
```python
myTuple = ("John", "Peter", "Vicky")
x = "#".join(myTuple)

print(x) # John#Peter#Vicky
a = ['1', '2', '3']
print(" => ".join(a)) # 1 => 2 => 3
```
