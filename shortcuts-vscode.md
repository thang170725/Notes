# shortcut shift
1. Shift + alt + f: Sắp xếp các dòng lệnh cho đẹp mắt, dễ nhìn.

# shortcut alt
2. Alt + ↑: Di chuyển cả dòng lệnh tại con trỏ lên phía trên.
3. Alt + E: mở tab edit vs code.
4. Alt + F: mở tab file trong vs code.
5. Alt + V: Mở tab view trong vs code.
    • Shift + alt + ↓: Copy xuống dòng liền kề, cả dòng tại con trỏ chuột.
    • Ctrl + alt + ↑: Chỉ định viết mấy dòng một lần
    • Ctrl + l: Bôi đen cả dòng tại con trỏ chuột.
    • Ctrl + d: Thêm vùng chọn vào kết quả find phù hợp.
    • Ctrl + /: Comment 1 dòng tại vị trí con trỏ chuột.
    • Ctrl + shift + k: Xóa một dòng tại vị trí con trỏ chuột
    • Ctrl + ]: Cả dòng lùi vào một kích thước khoản trắng =  1 tab

    • Ctrl + A: Chọn toàn bộ đoạn chữ.
    • Ctrl + C: Copy chữ.
    • Ctrl + F: tìm chữ.
    • Ctrl + H: Tìm và thay thế chữ.
    • Ctrl + N: Tạo một tài liệu mới.
    • Ctrl + O: Mở một file.
    • Ctrl + P:  Đi đến một file khác.
    • Ctrl + S: Lưu file.
    • Ctrl + V: paste file.
    • Ctrl + X: Cắt một đoạn chữ (cắt theo từng dòng).
    • Ctrl + Y: Redo text.
    • Ctrl + Z: Undo text.
    • Ctrl + Home: Đưa con trỏ lên đầu của file hiện tại.
    • Ctrl + End: Đưa con trỏ xuống cuối của file hiện tại. 
    • Ctrl + →: Con trỏ sang đúng bằng một chữ.
    • Ctrl + ←:  Đưa con trỏ sang trái đúng bằng một chữ.
